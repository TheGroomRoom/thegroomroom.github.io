# The Groom Room - Luxury Dog Grooming Website

A professional, fully responsive website for The Groom Room, a luxury dog grooming service in Stevens Point, Wisconsin.

## Features

- **7 Complete Pages**: Home, About, Products & Process, Certifications, Policies, Gallery, and Contact
- **Elegant Design**: Light green color scheme with Playfair Display and Lato fonts
- **Business Logo**: Professional branding displayed across all pages
- **Fully Responsive**: Works perfectly on mobile, tablet, and desktop devices
- **Functional Contact Form**: With validation and success messaging
- **Professional Layout**: Clean, spa-like aesthetic
- **Comprehensive Policies**: Detailed appointment and cancellation policies
- **Client Gallery**: Showcase of happy clients and grooming transformations

## Pages

1. **index.html** - Home page with hero section and services overview
2. **about.html** - Business story, philosophy, and what sets us apart
3. **products.html** - Premium products and detailed 8-step grooming process
4. **certifications.html** - Professional credentials and memberships
5. **policies.html** - Appointment policies, cancellation policies, and client guidelines
6. **gallery.html** - Photo gallery of happy clients and grooming transformations
7. **contact.html** - Contact information and message form

## How to View

Simply open `index.html` in any modern web browser, or serve the files using a local web server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000/index.html`

## Technologies Used

- HTML5
- CSS3 (with custom properties and flexbox/grid layouts)
- Vanilla JavaScript (for form handling)
- Google Fonts (Playfair Display & Lato)
